# Sistema de Gestión de Tareas - Práctica Git

Este es un proyecto colaborativo para practicar el flujo de trabajo con ramas.

## Stack Tecnológico (Simulado)
- **Frontend:** React / HTML5
- **Backend:** Node.js / Spring Boot
- **Database:** MySQL / MongoDB

## Guía de Instalación
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Disfrutar del código.
